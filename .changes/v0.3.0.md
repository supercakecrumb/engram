## v0.3.0 - 2026-07-19
### Added
* Bootstrap GitHub Actions CI (lint + build/vet/test) and a tag-triggered GitHub Release workflow.
* Lifecycle type (New/Introduced/Reviewing/Known) tracking an item's gating state orthogonally to CardState, with InFSRS and LifecycleFor helpers
* Introduction API (IntroOutcome, Scheduler.Introduce, Scheduler.SeedCardKnown) and daily-budget helpers (IntroItem, IntroConfig, NextIntroductions, RemainingIntroBudget) for gating new items before they enter FSRS review
* NextReview (queue.go): picks the next due card with no daily-new cap, for v2 callers whose novelty gate is upstream at introduction; NextDue and CountNewIntroduced are unchanged for backward compatibility
* quiz.Mode (Single/Set/Text) and set-choice exercises (quiz/set.go): KeySet, CanonSet, SetOption, SetExercise, GenerateSet, GradeIndex for grading by set-equality via index-based callbacks
* quiz.TextMatcher (quiz/text.go) for free-text ModeText grading: normalized (Unicode casefold + whitespace collapse) fuzzy match against an accept list within a Levenshtein tolerance. Exported Normalize and Levenshtein utilities, hand-rolled with no new dependency
* TipRequest gains a Mode field (additive; zero value = ModeSingle), so tip providers can tailor explanations to Set/Text exercises without breaking existing ModeSingle-only providers
